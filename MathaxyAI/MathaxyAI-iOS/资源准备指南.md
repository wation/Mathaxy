## 图片资源

### 1. 应用图标和启动屏

#### AppIcon（应用图标）

**位置**: `Mathaxy/Resources/Assets.xcassets/AppIcon.appiconset/`

**所需尺寸**:

| 尺寸 | 用途 | 格式 | 备注 |
|------|------|------|------|
| 1024x1024 | App Store | PNG | 用于应用商店展示 |
| 180x180 | iPhone (@3x) | PNG | iPhone 6s及以上 |
| 167x167 | iPad Pro (@2x) | PNG | iPad Pro 12.9英寸 |
| 152x152 | iPad (@2x) | PNG | iPad Air及以下 |
| 120x120 | iPhone (@2x) | PNG | iPhone 6/7/8 |
| 80x80 | iPhone (@2x) | PNG | iPhone 4/4s/5/5s/SE |
| 76x76 | iPad (@1x) | PNG | iPad 2及以下 |
| 40x40 | iPhone (@2x) | PNG | iPhone 通知图标 |
| 29x29 | iPhone (@2x) | PNG | iPhone 设置图标 |
| 20x20 | iPhone (@2x) | PNG | iPhone 通知小图标 |

**设计要求**:
- 使用明亮、友好的色彩
- 主题：数学、学习、儿童
- 建议元素：数字、星星、几何图形
- 背景色：#4A90E2（蓝色）或类似
- 确保在小尺寸下仍然清晰可辨

#### LaunchImage（启动屏幕）

**位置**: `Mathaxy/Resources/Assets.xcassets/LaunchImage.launchimage/`

**所需尺寸**:

| 尺寸 | 用途 | 格式 |
|------|------|------|
| 1125x2436 | iPhone X (@3x) | PNG |
| 1242x2208 | iPhone Plus (@3x) | PNG |
| 750x1334 | iPhone (@2x) | PNG |
| 2048x2732 | iPad Pro (@2x) | PNG |
| 1536x2048 | iPad (@2x) | PNG |
| 768x1024 | iPad (@1x) | PNG |

**设计要求**:
- 显示应用Logo和名称"Mathaxy"
- 使用与应用图标一致的配色
- 简洁、清晰的布局
- 考虑安全区域（刘海屏、灵动岛）

## 音效文件

### 1. 游戏音效

**位置**: `Mathaxy/Resources/Sounds/`

#### correct_answer.mp3

**用途**: 答对时的提示音

**技术要求**:
- 格式：MP3
- 时长：0.5-1秒
- 音量：适中
- 风格：欢快、积极
- 频率：44.1kHz
- 比特率：128kbps或更高

**建议音效**:
- 清脆的"叮"声
- 或简短的旋律片段

## 资源验证

### 1. 图片资源验证

**验证清单**:
- [ ] 所有图片都提供了1x、2x、3x三个分辨率
- [ ] 图片格式正确（PNG或PDF）
- [ ] 图片尺寸符合要求
- [ ] 透明背景图片使用PNG格式
- [ ] 图片文件大小合理（单张不超过500KB）
- [ ] 图片色彩符合设计要求
- [ ] 图片在小尺寸下仍然清晰

### 2. 音频资源验证

**验证清单**:
- [ ] 所有音频文件格式正确（MP3）
- [ ] 音频时长符合要求
- [ ] 音频音量适中
- [ ] 音频质量良好（无杂音）
- [ ] 音频文件大小合理（单个不超过1MB）
- [ ] 背景音乐支持循环播放
- [ ] 语音文件发音清晰、准确

### 3. 资源命名规范

**命名规则**:
- 使用小写字母和下划线
- 使用描述性的名称
- 避免使用特殊字符
- 保持命名一致性

**示例**:
- ✅ `home_background.png`
- ✅ `correct_answer.mp3`
- ❌ `HomeBackground.png`
- ❌ `correct-answer.mp3`

### 4. 资源文件大小控制

**建议限制**:
- 单张图片：不超过500KB
- 单个音效：不超过1MB
- 单个背景音乐：不超过5MB
- 单个语音文件：不超过500KB

## 注意事项

### 1. 版权问题
- 确保所有资源都有合法的使用权
- 使用免费或购买的商业授权资源
- 避免使用未经授权的素材

### 2. 性能优化
- 使用合适的图片格式和分辨率
- 压缩音频文件以减小体积
- 考虑使用矢量图（PDF）以减小体积

### 3. 多语言适配
- 确保所有语言版本的语音质量一致
- 语音发音标准、清晰
- 考虑不同语言的文化差异

### 4. 儿童友好
- 资源设计适合儿童审美
- 避免过于复杂或恐怖的元素
- 使用明亮、友好的色彩

### 5. 测试
- 在不同设备上测试资源显示效果
- 测试音频播放效果
- 确保资源加载速度合理
